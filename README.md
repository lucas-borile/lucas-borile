## Selected projects

### Resilium

A resilient communications platform designed for off-grid environments.

The project combines LoRa mesh networking, encrypted messaging, local services, offline-first workflows, device reputation, and cryptocurrency transactions relayed through connected nodes.

**Technologies:** Python, JavaScript, LoRa, Meshtastic, ESP32, Nano, Docker and local AI.

### Local AI over LoRa

An experimental interface connecting LoRa devices to locally hosted language models.

The system allows authenticated users to send compact commands and requests through a mesh network, which are interpreted by a local AI service and routed to automations or other users.

**Technologies:** Heltec LoRa 32, Meshtastic, Python, Ollama and local APIs.
